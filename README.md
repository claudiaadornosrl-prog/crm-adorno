# Claudia Adorno — CRM

App web para el equipo de vendedoras. Permite cargar y gestionar pedidos pendientes por local.

## Stack
- **Frontend**: HTML/CSS/JS vanilla (archivo único, sin build)
- **Base de datos**: Supabase (PostgreSQL)
- **Hosting**: GitHub Pages

## Archivos
- `index.html` — aplicación principal (vendedoras)

## Desarrollo local
Abrir `index.html` directamente en el navegador. Conecta en tiempo real con Supabase.

## Deploy
Cada `git push` a `main` actualiza automáticamente la app en GitHub Pages.
